# 🌍 TravelBuddy AI - Powered by Groq AI

TravelBuddy AI ek high-performance travel planner hai jo **Next.js**, **TypeScript**, aur **Tailwind CSS** se bana hai. Ye user ke location, mood, aur budget ke hisaab se personalized plans generate karta hai.

## ✨ Features
* **Lightning Fast AI**: Groq AI engine ka use kiya gaya hai fast responses ke liye.
* **Professional UI**: Glassmorphism design aur premium fonts ka use.
* **Cinematic Background**: Globe video background jo ek immersive experience deta hai.
* **Custom Typing Effect**: AI answers ek cool typing animation ke saath aate hain.

## 🛠️ Tech Stack
* **Frontend**: Next.js 16 (App Router)
* **Styling**: Tailwind CSS
* **AI Model**: Groq Llama-3
* **Deployment**: Vercel
